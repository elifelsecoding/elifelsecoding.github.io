Why coding?

Learning how to code is a crucial skill in the modern world, most high paying jobs require the use of coding. Learning how to code also improves your problem solving and critical thinking by challenging you to think outside the box! Finalay it's fun! Every class you learn something new and reinforce the newly learned material with a fun and interactive project. We teach Python, a fast growing intuitive programming language that is used by companies like Facebook and Google.

Q: Is coding hard to learn?

A: Nope! It is not hard when broken down into pieces, and demonstrated with projects.


Q: How long does it take to learn coding?

A: Learning a programming language like Python takes time and effort. We will do our best to make that time not only productive but also fun!


Q: What is a good age to start learning coding?

A: Any age is good to learn coding, but we recommend the student is between 8-14 years old.


Q: How will the classes be grouped?

A: We will group classes by age, skill, and of course requests to be with your friends!
